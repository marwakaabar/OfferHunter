Description Français
### "DealFinder" est une application Web de e-commerce full-stack qui utilise le MERN Stack (MongoDB, Express, React, Node.js) pour créer une plateforme conviviale et intuitive. L'application comprend des fonctionnalités telles que l'authentification et l'autorisation des utilisateurs, la navigation, la recherche et le filtrage des produits, ainsi que la fonctionnalité de panier d'achat. Il dispose également d'un tableau de bord d'administration où les administrateurs du site peuvent gérer les produits, les commandes et les utilisateurs.

### L'une des fonctionnalités uniques de "DealFinder" est la possibilité de rechercher des produits via une carte. Cette fonctionnalité permet aux utilisateurs de rechercher des produits en fonction de leur emplacement et de leur proximité avec le produit. Lorsqu'un utilisateur recherche un produit, il peut voir une carte qui montre l'emplacement du produit et les magasins à proximité qui vendent le produit. Les utilisateurs peuvent également filtrer les résultats en fonction de la distance, du prix et d'autres critères. Cette fonctionnalité est alimentée par l'API Open Street maps et s'intègre parfaitement avec le reste de l'application, offrant une nouvelle et innovante façon pour les utilisateurs de trouver des produits. La fonction de recherche de carte est un ajout précieux à l'application et démontre sa flexibilité et sa polyvalence dans la fourniture d'une solution de commerce électronique complète.

### Il convient également de mentionner que "DealFinder" était le projet de thèse de ma Licence. Ce projet m'a permis d'appliquer les connaissances et compétences acquises tout au long de mes études pour créer une solution pratique et innovante pour en E-commerce. Il m'a également fourni une expérience précieuse en gestion de projet, développement de logiciels et travail d'équipe. Le projet a été entièrement open-source et disponible sur GitHub pour que tout le monde puisse l'explorer, l'utiliser ou y contribuer.

### "DealFinder" a été construit en utilisant une variété de technologies modernes, notamment :

- MongoDB : une base de données NoSQL basée sur des documents utilisée pour stocker et gérer des données.
- Express : un cadre d'application web Node.js léger et flexible utilisé pour construire l'API côté serveur.
- React : une bibliothèque JavaScript utilisée pour construire l'interface utilisateur et fournir une expérience fluide et réactive.
- Redux : un conteneur d'état prévisible utilisé pour gérer l'état de l'application.
- Node.js : un environnement d'exécution JavaScript utilisé pour construire le code côté serveur et exécuter JavaScript en dehors du navigateur.
- OpenStreetMap : un service de carte open-source utilisé pour fournir une fonctionnalité basée sur la localisation à l'application.

- Git : un système de contrôle de version distribué utilisé pour suivre les changements dans le code source et collaborer avec les membres de l'équipe.
- GitHub: un service d'hébergement de référentiel Git basé sur le Web utilisé pour stocker et partager du code avec d'autres personnes.

Ces outils étaient essentiels pour le processus de développement et ont joué un rôle crucial pour garantir que l'application soit de haute qualité, efficace et évolutive.

English Description

### "DealFinder" is a full-stack E-commerce web application that utilizes the MERN (MongoDB, Express, React, Node.js) stack to create a user-friendly and intuitive ecommerce platform. The application includes features such as user authentication and authorization, product browsing, searching, and filtering, shopping cart functionality. It also features an admin dashboard where site administrators can manage products, orders, and users.

### One of the unique features of "DealFinder Ecommerce MERN Stack" is the ability to search for products through a map. This feature allows users to search for products based on their location and proximity to the product. When a user searches for a product, they can view a map that shows the location of the product and nearby stores that sell the product. Users can also filter the results based on distance, price, and other criteria. This feature is powered by the Open Street maps API and integrates seamlessly with the rest of the application, providing a new and innovative way for users to find products. The map search feature is a valuable addition to the application and demonstrates its flexibility and versatility in providing a comprehensive e-commerce solution.

### it's also worth mentioning that "DealFinder Ecommerce MERN Stack" was the thesis project of my Bachelor's degree. This project allowed me to apply the knowledge and skills I gained throughout my studies to create a practical and innovative solution for ecommerce. It also provided me with valuable experience in project management, software development, and teamwork. The project has been fully open-sourced and available on GitHub for anyone to explore, use, or contribute to.

### "DealFinder" was built using a variety of modern technologies, including:

- MongoDB: a document-based NoSQL database used to store and manage data.
- Express: a lightweight and flexible Node.js web application framework used to build the back-end API.
- React: a JavaScript library used to build the user interface and provide a seamless and responsive experience.
- Redux: a predictable state container used to manage the state of the application.
- Node.js: a JavaScript runtime environment used to build the server-side code and run JavaScript outside of the browser.
- OpenStreetMap: an open-source map service used to provide location-based functionality to the application.

### In addition to the technologies mentioned earlier, several tools were used in the development of "DealFinder", including:

- Visual Studio Code: a powerful and lightweight source code editor used for writing and editing code.
- Git: a distributed version control system used for tracking changes in the source code and collaborating with team members.
- GitHub: a web-based Git repository hosting service used to store and share code with others.
- Postman: a collaboration platform used for testing, debugging, and documenting APIs.
- NPM: a package manager used for installing and managing dependencies.
These tools were essential for the development process and played a critical role in ensuring that the application was of high quality, efficient, and scalable.
